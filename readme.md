# Feup

1) First clone this repo:

``` bash
git clone https://github.com/KlausSchaefers/feup-node-lecture-2021.git
```

2) Install Node dependencies
```bash
npm install
```

3) Run test cases to make sure everything works

```bash
npm run test
```

3) Launch Server
```bash
npm run serve
```
